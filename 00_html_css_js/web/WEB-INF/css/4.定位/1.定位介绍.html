CSS的定位： 1.static 静态定位 1）为position属性的 默认值 2）忽略 top、bottom、left、right、z-index 2.relative 相对定位 1）不脱离文档流的布局，只改变自身的位置，在文档流原先的位置遗留空白区域 2）定位的起始位置为此元素原先在文档流的位置 3）一般会和 float、margin、padding ... 一起使用 3.absolute 绝对定位 1）脱离文档流的布局，遗留下来的空间由后面的元素填充 2）定位的起始位置为最近的父元素(position不为static) 否则为body文档本身 说明： 1）起始位置看爸爸的位置，如果爸爸的定位是static，直接看到老祖宗body，然后通过 top、bottom、left、right 来改变直接的整体位置 2）不设置 top、bottom、left、right 起始位置走正常文档流 4.fixed 固定定位 1）类似于 absolute，但不随着滚动条的移动而改变位置 2）起始位置 不看任何人，只看浏览器窗口，通过 top、bottom、left、right，改变自己的整体位置 3）不设置 top、bottom、left、right 起始位置走正常文档流 5.sticky 类似 relative和fixed 的结合体 这几种定位(除static) 共同配合 下面5个属性 top、bottom、left、right、z-index(层级关系 body为0)
